Our house

by Michaela


Project background

The idea for this project was born, when I browsed the internet for a low-commitment/not professional space to display my own art, as well as see your everyday hobbyists arts and crafts. The idea in a nutshell was to create an app, where anyone and everyone could portray their art. The app would provide a safe space to showcase any handmade items, from paintings to sculptures to knitwork. In addition to a gallery, the app would include a chat/direct messaging, to allow the users swap art and chat about possible commissions.

User research

To determine what kind of features the users would like to have, an open card sorting was conducted. The focus group was first given an empty paper to write down whatever features and functions they would enjoy. After the open card sorting was completed, I gathered the most popular options to a list for the group to arrange from 1 to 10.

1. Easy to use
2. Safe for minors / copyrights
3. Free
4. Customizable
5. Low threshold to join, not a professional vibe
6. Chat
7. No payment info
8. Usable on mobile & tablet
9. Anonymity as an option
10. Banner/Slider

Either a diary or a survey for feedback should be filled by the group after few weeks of use.

Summary of findings

The most wanted features were a gallery that is easy to add in, and easy to organize. A chat that would require logging in, in order to avoid trolls and to create a more safe environment. The focus group also wished for the app to be customizable, to better suit each users personality. One of the most requested feature was a rating system, much like defining the condition of an used product. Least wanted features were a rating system and an option to sell/buy directly, as the users did not want to add pricing to their work.

Problems & Solutions

Looking into the user research, some problems were surfacing. If the user is a minor, will they see inappropriate content? The app should have an option to mark content as sensitive and/or to report any unmarked or otherwise unruly content that violates the guidelines. In case the user is a minor, the users parent should have the option to automatically hide all marked content. The app should also provide a (c) Copyright to the artists, and have a Terms & Conditions agreement to prevent any plagiarism.




User Stories


As a pottery enthusiast I want to swap my little pots and vases with other hobbyists, so that I can make room for new pots and vases!

As a painter I want to showcase my art so that people can see it without me having to rent a space.

As an arts & crafts person, I want to share my work and thoughts with other likeminded people, so that I can make new friends.

As someone who experiments with all kinds of art, I want to be able to not excell at everything, so that there is no pressure in creating.

As a mother of an artsy child, I want to save my kids artwork online, so that I can keep them with me.

As a hobbyist, I want to make custom keycaps for others, so that I don’t have to make caps for my own keyboard only.

As an artist, I want to browse other people’s art so that I can gain inspiration.

As a knitter, I want to swap patterns so that I can learn new techniques.

As a painter I want to make commissions that are not expected to be perfect, so that I can make a little extra cash.

As a shy person, I want to showcase my craft anonymously, so that I can gain confidence.
